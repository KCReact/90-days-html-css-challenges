# Paragraphs
<p>We can literally write any paragraph and enclose it in between these p tags.
    The text could even span multiple lines too.</p>

-> <strong> Any text written in between these tags would get highlighted meaning it looks bolder than the remaining text.</strong>
-> <em> em tag is used to italicize the text.</em>
