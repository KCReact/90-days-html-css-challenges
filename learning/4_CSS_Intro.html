CSS - Cascading Style Sheets -> CSS is a stylesheet language used to describe the visual presentation and layout of a document which is written in a markup language HTML. -> CSS can be applied to a HTML file in 3 ways : 1) Internal -> Styles are applied when they are written in between tags in the tag of HTML document. 2) Inline -> Styles would be applied directly to the tags with attribute 'style' which can be filled with any tag existing in HTML. For instance,

Red Coloured Text with Font size of 30 px

Here the text will have background color as red and the font-size of it's text would be 30pixels. 3) External -> Styles are combined for the whole page based on the tags or selectors (class , id , attribute) and written separately in another page. -> The file is linked to the desired HTML page using 'link' tag (), provided the styles.css should reside in same folder where the HTML file is already existing.
