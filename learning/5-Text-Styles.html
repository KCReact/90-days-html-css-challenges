-> In CSS, we can style the text with a lot of properties, out of them we have touched the surface using the below properties, 1) font-family : There are various fonts which are mainly comprised into either serif or sans-serif root fonts. 2) font-weight : The weight indicates whether the text should be bold or not and also we can set the boldness with values such as 200,400,600,800,900. 3) font-style : The values mainly include dotted, italic , none etc. 4) font-size : Basically measured in px (pixels) on the screen, though there are other units such as rem or em too. 5) text-align : Alignment of text according to the place it is existing in. 6) text-transform : Let's us decide the case of text (lower/upper). 7) line-height : Governs the spacing between the lines which wrap one under the other as a result of huge amount of info on the page.
