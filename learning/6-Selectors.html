Selectors in CSS -> CSS selectors are patterns used to "find" and select the specific HTML elements you want to style. There are different type of selectors in CSS, -> Type Selector: Selects elements based on their HTML tag name (e.g., p, h1, div). -> ID Selector (#) : Targets a single unique element with a specific id attribute. IDs should only be used once per page. -> Class Selector (.) : Targets all elements with a specific class attribute. Classes can be reused on multiple elements. -> Descendant Selector (space): Selects all elements inside a specified element (e.g., div p targets every
