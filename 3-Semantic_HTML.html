# Semantic Tags -> Semantic Tags are essential for giving a meaning to the structure of an HTML document. -> It is also used by screen readers, SEO to properly read the content of a website or webpage. List of semantic tags ->
- Header indicates the topmost content of the webpage. ->
- Navigation tag generally contains the navigation links to other pages. ->
- Article is one large entity where there would be lots of infomration in the form of sections , lots of side bars and also lots of other main info. ->
- A large article can be divided into sub parts using a section tag which can specify what part of page is specifically displying certain info. ->
- aside section generally encloses the stuff which is not much important and it's kinda like side bar. ->
- Footer indicates the bottom most content of the webpage.
